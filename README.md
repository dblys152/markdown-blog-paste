# MD2Blog

Markdown으로 작성한 글을 블로그 에디터에 붙여넣기 좋은 HTML로 변환하고, HTML/PDF 파일로 저장할 수 있는 웹 도구입니다.

서비스 바로가기: https://md2blog.pages.dev

## 주요 기능

- Markdown 파일 업로드 및 미리보기
- 미리보기 HTML을 클립보드에 복사
- HTML 파일 다운로드
- PDF 파일 다운로드
- 블로그 에디터용 빈 줄 보정
- 네이버 블로그용 코드블록 변환
- HTML 미리보기/다운로드 결과의 플로팅 섹션 내비게이션

## 변환 모드

| 모드 | 설명 |
|---|---|
| 기본 변환 | Markdown을 기본 HTML로 변환합니다. |
| 빈 줄 추가 | 블로그 에디터에서 문단 간격이 유지되도록 빈 줄을 추가합니다. |
| 네이버 블로그용 | 네이버 에디터에 맞게 코드블록과 빈 줄 처리를 보정합니다. |

## 개발

```bash
npm install
npm run dev
```

개발 서버가 실행되면 브라우저에서 `http://127.0.0.1:5173/`을 열어 사용합니다.

## 빌드

```bash
npm run build
```

정적 파일은 `dist/`에 생성됩니다.

## Python CLI

`python-cli/`는 웹 앱 이전에 만든 CLI 버전입니다.

현재 주요 기능은 웹 앱을 기준으로 개발하고 있으며, CLI는 로컬에서 단일 Markdown 파일을 HTML로 변환할 때 사용할 수 있습니다.
